Recent Work

Pro Shop

A full-stack MERN website on which the user can view various products before putting them into their basket and buying. Inspiration for this site came from the frustration of trying to order from sites which overloaded the page with excessive and contradictory information. I wanted to create a slimmed down site which was intuitive for the user to navigate, provided relevant information, and helped them to complete their purchase as easily as possible.

Github Finder

A React website which uses the GitHub API to search through all the users on GitHub by their username. All the users which match the search can be clicked on and a user page displays relevant information about the user as well as links to the user's GitHub. The inspiration for this was to take data from an API and manipulate as much of that data as possible to create a website based almost entirely on this data. It serves as a model for creating other websites around data from APIs.
